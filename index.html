Bem-vindo ao Aprenda JavaScript.

Sem enrrolação, vamos brincar um pouco com JavaScript.

1° passo - vamos adicionar no HTML um evento na tag input 'ler' chamada 'escrever' que será executada quando o usuário digitar.

assim:
<input id='ler' onkeydown="escrever()">

2° passo - Agora na parte do JavaScript vamos criar a função 'escrever'.

assim:
function escrever(){

}

3° passo - entre '{' e '}' vamos escrever o que ela vai fazer.

Ela vai pegar o que será escrito na caixa de texto(tag a input) e colocar no paragrafo(tag a p)

assim:
let txt = document.querySelector("#ler").value;
document.querySelector("p").innerText = txt;

4° passo - Agora só clicar no botão 'Executar' e digitar qualque coisa no input.
